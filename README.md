# Auditing Fairness in AI-Based Resource Allocation

## Overview

This study evaluates fairness in AI-based resource allocation using the Adult Income Dataset, analyzing accuracy, bias, explainability, and allocation strategies.

Results reveal key fairness–utility trade-offs and support the development of more equitable AI systems.

## Research Questions

- RQ1: How do different machine learning models compare in terms of fairness and predictive accuracy in resource allocation decisions?
- RQ2: Which features most strongly influence AI-based allocation decisions, and how explainable are these outcomes?
- RQ3: How does historical bias in training data affect prediction outcomes across gender and racial groups?
- RQ4: Which fairness metric provides the most suitable balance between equity and model performance?
- RQ5: How do fairness-aware allocation strategies impact the trade-off between utility and equitable resource distribution?
